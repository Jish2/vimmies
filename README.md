# vimmies

my favorite vim motions that i'll never remember

ranked by usage: most 🔥 👍 😑 👎 🗑️ least

## normal

### `<C-i/o>` 🔥

move positions in jump list

## visual

### `o` 👍

move to start or end of visual selection

## commands

### `:'<,'>norm {motion}` 👎

executes `{motion}` on given visually selected lines
